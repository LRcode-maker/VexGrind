# VexGrind
Vex educationnal website.
VEX Robotics is one of the most competitive STEM programs on the planet. But for too long, the best strategies, code, and build knowledge were locked away in private Discord servers and top-team notebooks that never got shared.

VexGrind changes that. We believe every team can work together to build this community and create the greatest robots Vex has ever seen

We publish everything: auton routines, CAD files. No paywalls. No gatekeeping. Just the information to help you succeed
